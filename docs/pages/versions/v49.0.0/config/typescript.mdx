See more information about **tsconfig.json** in the [TypeScript guide](/guides/typescript/).

Expo recommends the following base `tsconfig.json`:

```
{
  "extends": "expo/tsconfig.base",
  "compilerOptions": {
    "strict": true,
  },
  "include": [
    "**/*.ts",
    "**/*.tsx",
    ".expo/types/**/*.ts",
    "expo-env.d.ts"
  ]
}
```
